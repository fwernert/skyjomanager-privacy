Politique de Confidentialité

Dernière mise à jour : 21 mai 2026

1. Introduction

Skyjo Game Manager est une application mobile gratuite développée par Florent WERNERT. Cette politique de confidentialité décrit comment l'application traite vos données.

2. Données collectées

Aucune donnée personnelle n'est collectée, transmise ou partagée.

L'application stocke uniquement les données suivantes localement sur votre appareil :

  • Prénoms des joueurs (saisis par vous)
  • Scores et historique des parties
  • Préférence de langue

Ces données ne quittent jamais votre appareil. Elles ne sont envoyées à aucun serveur externe.

3. Stockage des données

Toutes les données sont stockées via AsyncStorage (stockage local de l'appareil). La suppression de l'application entraîne la suppression définitive de toutes les données. Vous pouvez également réinitialiser les données depuis l'application (bouton « Réinitialiser toutes les données »).

4. Services tiers

L'application n'utilise aucun service tiers de suivi, d'analyse ou de publicité. Aucun SDK tiers ne collecte de données.

5. Permissions

L'application ne requiert aucune permission spéciale (pas d'accès à la caméra, au microphone, aux contacts, à la localisation, etc.).

6. Enfants

L'application est adaptée à tous les âges. Aucune donnée personnelle n'étant collectée, elle est conforme au RGPD et au COPPA.

7. Modifications

Cette politique peut être mise à jour. Les changements seront publiés sur cette page avec la date de mise à jour.

8. Contact

Pour toute question concernant cette politique :
wernertflorent@gmail.com

Privacy Policy

Last updated: May 21, 2026

1. Introduction

Skyjo Game Manager is a free mobile application developed by Florent WERNERT. This privacy policy describes how the application handles your data.

2. Data Collected

No personal data is collected, transmitted, or shared.

The application only stores the following data locally on your device:

  • Player names (entered by you)
  • Game scores and history
  • Language preference

This data never leaves your device. It is not sent to any external server.

3. Data Storage

All data is stored via AsyncStorage (device local storage). Uninstalling the application permanently deletes all data. You can also reset data from within the app ("Reset all data" button).

4. Third-Party Services

The application does not use any third-party tracking, analytics, or advertising services. No third-party SDK collects data.

5. Permissions

The application requires no special permissions (no access to camera, microphone, contacts, location, etc.).

6. Children

The application is suitable for all ages. Since no personal data is collected, it complies with GDPR and COPPA.

7. Changes

This policy may be updated. Changes will be published on this page with the update date.

Datenschutzerklärung

Letzte Aktualisierung: 21. Mai 2026

1. Einführung

Skyjo Game Manager ist eine kostenlose mobile Anwendung, entwickelt von Florent WERNERT. Diese Datenschutzerklärung beschreibt, wie die Anwendung mit Ihren Daten umgeht.

2. Erhobene Daten

Es werden keine personenbezogenen Daten erhoben, übermittelt oder weitergegeben.

Die Anwendung speichert nur die folgenden Daten lokal auf Ihrem Gerät:

  • Spielernamen (von Ihnen eingegeben)

Die Anwendung verwendet keine Tracking-, Analyse- oder Werbedienste von Drittanbietern. Kein Drittanbieter-SDK erhebt Daten.

Die Anwendung benötigt keine besonderen Berechtigungen (kein Zugriff auf Kamera, Mikrofon, Kontakte, Standort usw.).

Die Anwendung ist für alle Altersgruppen geeignet. Da keine personenbezogenen Daten erhoben werden, entspricht sie der DSGVO und dem COPPA.
